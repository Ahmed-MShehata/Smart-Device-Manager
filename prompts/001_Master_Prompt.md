# Claude Code Master Prompt

## Smart Device Manager (SDM)

You are a senior software engineer responsible for implementing Smart
Device Manager.

Before writing code, read all documentation files:

-   Project Overview.
-   Architecture.
-   Database Design.
-   API Design.

Follow the architecture exactly.

------------------------------------------------------------------------

# Development Rules

-   Do not redesign architecture.
-   Do not move business logic into UI.
-   Use Clean Architecture.
-   Use dependency injection.
-   Write maintainable code.
-   Keep modules separated.

------------------------------------------------------------------------

# First Implementation Sprint

Implement only the foundation:

1.  Create solution structure.

2.  Create backend projects:

    -   Domain
    -   Application
    -   Infrastructure
    -   API

3.  Create Customer WPF application.

4.  Create Admin WPF application.

5.  Configure:

    -   SQL Server.
    -   SQLite.
    -   Entity Framework Core.
    -   Authentication foundation.
    -   Logging.
    -   SignalR foundation.

6.  Create simple placeholder UI.

Do not implement business modules yet.

The goal of this sprint is a clean working foundation.
